Panorama to Cubemap

Reference: https://stackoverflow.com/questions/29678510/convert-21-equirectangular-panorama-to-cube-map

Core ideas:

  • View panorama as the equirectangular projection of sphere, such that spherical coordinate can be directly used to query its pixel value
  • Map from target to source (inverse transformation), by finding the spherical representation of cartesian coordinate in target image
  • Perform sampling and interpolation

Details

Spherical coordinates to Cartesian coordinate convertion

{x=rsin⁡θcos⁡ϕy=rsin⁡θsin⁡ϕz=rcos⁡θ{\begin{cases} x= r\sin\theta\cos\phi \\ y = r\sin\theta\sin\phi \\ z=r\cos\theta \end{cases}}⎩⎨⎧​x=rsinθcosϕy=rsinθsinϕz=rcosθ​

Where

{r=10<θ<π−π4<ϕ<7π4\begin{cases} r=1 \\ 0<\theta<\pi \\ -\frac{\pi}{4} < \phi < \frac{7\pi}{4} \end{cases}⎩⎨⎧​r=10<θ<π−4π​<ϕ<47π​​

Project to sides

Let x=1x = 1x=1 such that a=1sin⁡θcos⁡ϕa = \frac{1}{\sin\theta\cos\phi}a=sinθcosϕ1​ So

y=tan⁡ϕ,z=1tan⁡θcos⁡ϕy = \tan\phi, z = \frac{1}{\tan\theta\cos\phi}y=tanϕ,z=tanθcosϕ1​

Then

ϕ=atan2(y),θ=atan2(1zcos⁡ϕ)\phi = atan2(y), \theta = atan2(\frac{1}{z\cos\phi})ϕ=atan2(y),θ=atan2(zcosϕ1​)

Project to top

Let z=1z = 1z=1 such that a=1cos⁡θa = \frac{1}{\cos\theta}a=cosθ1​ So

x=tan⁡θcos⁡ϕ,y=tan⁡θsin⁡ϕx = \tan\theta\cos\phi, y = \tan\theta\sin\phix=tanθcosϕ,y=tanθsinϕ

Then

ϕ=atan2(yx),θ=atan2(x2+y2)\phi = atan2(\frac{y}{x}), \theta = atan2(\sqrt{x^{2} + y^{2}})ϕ=atan2(xy​),θ=atan2(x2+y2​)

Python sample code

# left, front, right, back, top, bottom
offset = [0, 0.25, 0.5, 0.75, 0, 0]

face_id = 0

for w in range(edge):
    for h in range(edge):
        
        # map to range (-1, 1)
        u = (w/edge - 0.5)*2
        v = (h/edge - 0.5)*2
        
        if face_id == 4:
            phi = pi - atan2(v, u)
            theta = atan2(sqrt(v*v+u*u), 1)
        elif face_id == 5:
            phi = -pi + atan2(v, u)
            theta = -atan2(sqrt(v*v+u*u), 1)
        else:
            phi = atan2(u, 1)
            theta = atan2(-1, v*cos(phi))
        
        x = phi/(2*pi)
        y = theta/pi
        
        # TODO: use bilinear interpolation
        x_idx = int(width*(x + offset[face_id]))
        y_idx = int(height*y)
        
        face_pixel[w, h] = panorama_pixel[x_idx, y_idx]
