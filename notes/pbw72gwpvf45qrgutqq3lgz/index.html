Cholesky Decomposition

Reference: https://www.math-linux.com/mathematics/linear-systems/article/cholesky-decomposition

Symmetric positive definite A\bm{A}A can be decomposed into A=LLT\bm{A} = \bm{L}\bm{L}^TA=LLT, where L\bm{L}L is lower triangle matrix

Application

To solve Ax=b\bm{A} \bm{x} = \bm{b}Ax=b, first solve Ly=b\bm{L} \bm{y} = \bm{b}Ly=b then y=LTx\bm{y} = \bm{L}^T \bm{x}y=LTx. Note linear system with lower/upper triangle matrix can be easily solve with forward substitution / backward substitution
