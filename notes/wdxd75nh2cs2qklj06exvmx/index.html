Signed Distance Function

Signed distance function is a function defined in x∈R3\bm{x} \in \mathbb{R}^3x∈R3, such that the function value f(x)f(\bm{x})f(x) equals to the signed distance from point x\bm{x}x to the closest surface. To fit it, we need:

  • A set of surface points xi∈R3\bm{x}_i \in \mathbb{R}^3xi​∈R3 (zero-level set f(xi)=0f(\bm{x}_i) = 0f(xi​)=0)
  • Normal vectors ni∈R3\bm{n}_i \in \mathbb{R}^3ni​∈R3 for those points

Find local plane

Reference: https://www.graphics.rwth-aachen.de/media/papers/p_Pau021.pdf

For each point x\bm{x}x, we pick the n−1n-1n−1 nearest neighbour points. The objective is to fit a plane for these nnn points, such that the sum of distance for each point to the plane is minimized. The distance of point to surface can be modeled as Dot Product with surface normal n\bm{n}n, given a point c\bm{c}c the plane pass through.

min∑i=1n((xi−c)Tn)2min \sum_{i=1}^{n} ((\bm{x}_i - \bm{c})^T\bm{n})^2mini=1∑n​((xi​−c)Tn)2

Note that it is equivalent to Principle Component Analysis, by performing eigenanalysis on the covariance matrix (Cov∈R3×3\mathrm{Cov} \in \mathbb{R}^{3 \times 3}Cov∈R3×3) of those points. We pick (±)n(\pm)\bm{n}(±)n as the eigenvector that corresponds the smallest eigenvalue (We want to minimized the variance as opposed to PCA)
