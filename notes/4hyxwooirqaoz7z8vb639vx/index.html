s.array(); // Array

Partial reduction

  • rowwise(): treat as 1 x cols()
  • colwise(): treat as rows() x 1

Example 1

V: n x 3 (MatrixXd)
TT: 4 x 4 transformation matrix

(V.rowwise().homogeneous()*TT).rowwise().hnormalized())

Example 2

P: n x 3 (MatrixXd)
v: 1 x 3 (RowVector3d)

P.rowwise() - v: n x 3

Example 3

P: n x 3 (MatrixXd)
v: n (VectorXd)

(P.array().colwise() * v.array()).matrix()
