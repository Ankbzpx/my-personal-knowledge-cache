Colormap

Implementation

colormap

palette p: 256 x 3 (RGB8) array
factor f: [0, 1]

mapped_f = f * (256 - 1)
a = floor(f)
b = ceil(f)

# apply linear interpolation for rgb
c = mix(p[a], p[b], mapped_f - a)
